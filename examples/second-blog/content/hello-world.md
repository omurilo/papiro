# O Início de uma Nova Jornada

Se você está lendo isso, significa que o comando `init` funcionou perfeitamente e o motor do **Papiro** já está rodando! 

Este é um gerador de sites estáticos focado em simplicidade, velocidade e na beleza da escrita em texto puro. Tudo o que você precisa fazer é escrever em Markdown e deixar que o Go faça o resto.

## O que você pode fazer aqui?

Como usamos o padrão Markdown, você tem total liberdade para formatar seus textos de forma rápida:

* Criar **textos em negrito** para dar ênfase.
* Usar *itálico* para pensamentos ou termos estrangeiros.
* Fazer listas organizadas, como esta.

Se precisar citar alguém importante, o design clássico cuida disso:

> "A simplicidade é o último grau de sofisticação." 
> — Leonardo da Vinci

### Suporte a Código

E como todo bom desenvolvedor, você pode compartilhar seus trechos de código facilmente. O Papiro já deixa tudo bem formatado:

```go
package main

import "fmt"

func main() {
    fmt.Println("O Papiro é rápido demais!")
}
```
